leetcode 9.回文数

题目

判断一个整数是否是回文数。回文数是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

题目链接

参考文章：

极简数学解法

思路

首先想到的是转换成字符串，判断是否回文，这样效率不是很高。因此，要换个办法。

有两种方法可以解决这个问题。首先可以求取给定数的逆序，判断与原数是否相等。还可以取数字的首位和末位依次比较。下面的三种解法均基于这两个办法。

解法

易得：当x小于0且是10的倍数时一定不为回文数

​ 当x小于10且大于0时一定是回文数

解法1：取数字的首位和末位依次比较。

首先需要解决的问题是如何在不转换成字符串的情况下求得数字位数。

设数字为x，数字位数为n，公式 $n=10^{\left \lfloor\log_{10}{x}\right\rfloor} $

取最后一位：x%10，取第一位：x/n

每次循环去掉首位与末尾：x=(x%n)/10

完整代码

bool isPalindrome(int x)
{
    //数学解法
    if (x < 0 || (x % 10 == 0 && x != 0))
        return false;
    if (x < 10)
        return true;

    int n = log10(x);
    int nn = pow(10, n); //求x的数量级
    while (x > 0)
    {
        if (x / nn != x % 10)
            return false;
        x = (x % nn) / 10;
        nn /= 100;
    }
    return true;
}

解法2 求取逆序数字

直接上代码

bool isPalindrome(int x)
{
    //反转全部数字
    if (x < 0 || (x % 10 == 0 && x != 0))
        return false;
    if (x < 10)
        return true;

    long ans = 0;
    int a = x;
    while (a != 0)
    {
        ans = ans * 10 + a % 10;
        a /= 10;
    }
    return x == ans;
}

此处有个问题：32位整数最大为2147483647，如果将其反转，则会发生溢出。所以ans采用long来存储。

解法3 求取逆序一半的数字

回文数的特点是具有对称性。我们完全可以反转一半的数字就能判断是否为回文数。

根据数字位数的不同可分为两种，奇数位和偶数位

举个例子：123321与1234321

123321是123与123进行比较；1234321是1234与123进行比较。可见奇数位需要去掉求得的逆序数的最后一位

循环的终止条件，当求得的逆序数比原数字大或相等时停止

bool isPalindrome(int x)
{
    //反转一半数字
    if (x < 0 || (x % 10 == 0 && x != 0))
        return false;
    if (x < 10)
        return true;

    int ans = 0;
    int a = x;
    while (ans < a)
    {
        ans = ans * 10 + a % 10;
        a /= 10;
    }
    return a == ans || a == ans / 10;
}
